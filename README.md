# README — Shab TraceBrowser

## What the project does
Shab TraceBrowser is a web browser application developed for security testing and performance monitoring. It provides various features to help users analyze web pages, including network monitoring, security analysis, and performance metrics.

## Why the project is useful
This browser is designed to assist developers and security professionals in testing web applications. It offers tools to block resources, monitor network traffic, check permissions, and analyze web page security.

## How users can get started with the project

### Requirements
- .NET 8.0 Runtime
- Windows operating system (x64)

### Installation
1. Download the latest release from the GitHub releases page.
2. Extract the contents of the zip file.
3. Run the `TraceBrowser.exe` executable.

### Usage
- **Navigation**: Enter a URL in the address bar and click "Go".
- **Blocking Resources**: Use the checkboxes to block images, JavaScript, CSS, fonts, or third-party resources.
- **Network Monitoring**: Check the "Network" tab to view HTTP requests, response headers, and network performance.
- **Security Analysis**: The "Security" tab provides information about TLS versions, certificates, and security headers.
- **Permissions**: Use the "Allow All Permissions" and "Grant Extra Access" checkboxes to configure browser permissions.
